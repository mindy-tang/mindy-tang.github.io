hexo博客搭建记录

安装node.js

打开iterm终端，切换到root用户进行操作

sudo su

查看node版本：node -v
查看npm版本：npm -v
通过npm安装cnpm：npm install -g cpm –registry
安装hexo框架：cnpm install -g hexo-cli
查看hexo版本： hexo -v

使用hexo搭建blog

创建新项目文件夹：mkdir blog
在blog文件夹中初始化一个博客： hexo init

将blog部署到github上

在github中新建blog仓库：（必须与github账号同名.github.io）
安装部署插件：cnpm install –save hero-deployer-git
修改_config.yml文件：vim _config.yml
deploy：
type: git
repo: github中的blog仓库地址
branch: master
部署到远端：hexo d

切换主题

在github中找到喜欢的主题项目 clone到自己的博客theme文件夹中
修改_confg.yml 中的theme: 主题名
hexo clean 清理
hexo g
hexo s 重启项目
hexo d 推到远端
